<Info>
  The TypeScript SDK currently ships the OpenTelemetry transport (`BentoLabsSpanProcessor` + `BentoLabsTraceExporter`). The higher-level analytics surface (`trackAi`, `begin`, decorators) lands in a later release. For Node-side agent tracing today, wire the span processor into your tracer provider as shown in [Usage](/typescript/usage).
</Info>

## Requirements

- Node.js 18 or higher
- An OpenTelemetry tracer provider (we use `@opentelemetry/sdk-trace-node` in examples)

## Install

<InstallTypeScript />

`@opentelemetry/api` and `@opentelemetry/sdk-trace-base` are peer dependencies. Bring whichever versions match your tracer setup. `@opentelemetry/sdk-trace-node` is only needed if you're standing up a fresh `NodeTracerProvider`. If you already have a tracer configured, you don't need it.

## Set your API key

<ApiKeySetup />

Or pass it explicitly to the span processor:

```ts
new BentoLabsSpanProcessor({ apiKey: "bl_pk_..." });
```
